ENTRADA:

- items: lista de numeros enteros a modificar

SALIDA:

- items: lista modificada

ALGORITMO:

N = len(items)

for i in range(0, N - 1):
    min_idx = i

    for j in range(i + 1, N):
        if items[j] < items[min_idx]:
            min_idx = j

    if min_idx != i:
        temp = items[i]
        items[i] = items[min_idx]
        items[min_idx] = temp

return items
